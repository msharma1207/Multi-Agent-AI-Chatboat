# Phased Delivery Plan

1. Foundation scaffold
   - Backend API, frontend shell, tests, Docker
2. Free-tier AI integration
   - Ollama + local models for development
   - Groq and OpenRouter for low-cost inference
   - Gemini and HuggingFace options via configuration
3. Knowledge and agents
   - Qdrant, embeddings, RAG, multi-agent orchestration
4. Production hardening
   - PostgreSQL, Redis, auth, RBAC, observability, deployment
