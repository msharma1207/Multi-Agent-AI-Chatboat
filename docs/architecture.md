# Architecture Overview

## Phase 1 (current)
- FastAPI backend with auth and chat endpoints
- LangGraph orchestration hooks
- Next.js frontend shell
- Docker compose scaffold

## Phase 2
- PostgreSQL + Redis integration
- JWT auth with refresh flow
- Qdrant-backed knowledge base
- Agent workflows with supervisor/planner/research agents

## Phase 3
- File ingestion, chunking, embeddings, RAG retrieval
- Streaming chat UI and websocket support
- AWS deployment, CI/CD, monitoring
